# AWS S3 Buckets with Terraform

This project demonstrates how to provision and manage Amazon S3 buckets using Terraform. It covers configuring AWS CLI credentials, writing Terraform configuration files, initializing the workspace, and deploying cloud resources using Infrastructure as Code.

## What I Built

- Installed and configured Terraform on a local machine  
- Set up AWS CLI with IAM access keys for programmatic authentication  
- Wrote a Terraform configuration defining an S3 bucket and public access block  
- Initialized the Terraform workspace and generated an execution plan  
- Applied the configuration to deploy the S3 bucket in AWS  
- Demonstrated repeatable, automated cloud provisioning using IaC principles
